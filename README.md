# Personal Project: Food Order App

Made with JavaScript, React.js, HTML/CSS. <br/>

React hooks used:
 - useState
 - useEffect
 - useContext
 - useRef
 - useReducer

### To run this React app:
 - Clone the repository
 - run 'npm install' in the terminal
 - run 'npm start' in the terminal
